⚠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PORT IN USE?
  → Use: PORT=3001 npm start

DEPENDENCIES NOT INSTALLING?
  → Run: npm cache clean --force
  → Then: npm install

PAGES NOT LOADING?
  → Press F12 to open console
  → Check for errors
  → Refresh browser (Ctrl+R)

CLEAR BROWSER DATA?
  → Press Ctrl+Shift+Delete
  → Clear browsing data
  → Refresh page
